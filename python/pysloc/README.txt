Pysloc
-------

pysloc is a simple utility for counting source lines of code for Python source files,
featuring a proper abuse of the global keyword :)

Usage
-----
Clone the repo.
cd to pysloc folder on cmd/terminal
run:
    python pysloc.py filename
filename is the python file name.

Dependencies
-------------
switches -> pip install switches.py

Supported Python Versions
-------------------------
Python 3.6+
